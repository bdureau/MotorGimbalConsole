Bear Altimeter Motor Gimbal Console help file

This is a quick help file to use your Android Bear altimeter Motor Gimbal console. The Android console is a very convenient way to re-configure your Gimbal when you are on the field.


First of all you need to make sure that your Gimbal is compatible (ie: the correct firmware has been loaded).

Secondly make sure that the bluetooth module is connected on the Gimbal.

Thirdly you need to go to your android device settings, go to the bluetooth configuration and search for the HC-05 or HC-07 bluetooth module. When you have find it enter the pairing code which by default is 1234.

!!! CAUTION !!!!

You might want to clearly name each bluetooth device paired in your phone to make sure that you configure the correct Gimbal.

 

Then turn on the Gimbal and wait until the initialisation is complete.

Then start the application on your Android device, click on the show paired devices and finally select the HC-05 or HC-06 device. It now attempt to connect to the Gimbal.

If it says connected, you are now in and can use the Console to configure your Gimbal or read the flights. If it cannot connect then make sure that you have the correct bluetooth device selected or that the Gimbal is on.

 

Using the Gimbal Console

Gimbal main screen

On the main screen you have a number of buttons, let's go and describe each one.

[Gimbal Config]

Read the Gimbal setting and allow you to modify them.

 

[Flights]

Read all the flights that have been recorded, plot them or even play them back.

 

[Reset]

Gives the ability to restor to factory settings the Gimbal config. It also allow to clear all flightsor just the latest flights.

 

[Flash Firmwares]

Gives the ability to flash the Gimbal with the latest firmware and make sure that it is compatible with the Gimbal console

 

[Connect/Disconnect]

Connect or Disconnect from the Gimbal

 

Altimeter main screen menu

From the menu you can access this help file, an about screen that will give you information about the console and its version and finally you have the option to customize the look and feel of this application.

 

 

Supported altimeters for this console are:

- Gimbal (MPU6050)

- Gimbal (BNO055)
