Configuration d'un module Bluetooth

De manière à pouvoir configurer un module bluetooth pour connecter un altimetre à votre telephone vous aurez besoin de le configurer pour qu'il soit à la meme vitesse que votre altimètre .

Cet ecran vous aidera à configurer un module bluetooth de maniere à pouvoir l'utiliser avec votre carte.

Les modules comme le HC05, HC06 etc ... devraient fonctionner

Pour programmer le module vous avez besoin d'un cable TTL

un adaptateur USB (USBC ou micro USB suivant votre apareil Android )

un adaptateur jst et des cables jst

Notez que ces modules sont en général pré-configurés à 9600 bauds. Cependant votre altimetre par defaut est configuré à 38400 bauds, il faudra donc vous assurer de vous connecter à la bonne vittesse.

Attention: Assurez vous de ne pas débrancher le module lorsque vous faites ce qui suit

Charger la configuration du module

Connectez le module à votre appareil Android en utilisant un cable TTL et un adaptateur USB compatible avec votre telephone en micro USB ou USB type C.

Vous auraez besoin de choisir le module bluetooth que vous avez. Si vous ne savez pas lequel vous avez choisissez celui par defaut ou tout autre module compatible. Si vous ne pouvez utilisez aucun des modules envoyez moi la reference du module et/ou des photos du module.

Ensuite cliquez sur [Charger cfg], cela tentera de se connectez au module en utilisant la vitesse pour defaut de 38600 baud. Si cela ne fonctionne pas essayez encore, cela utilisera differentes vitesses jusqu'à ce qu'il se connecte. Cela chargera la configuration du module de manière à ce que vous puissiez la changer

Sauvegarder la configuration du module

Changez les parametres que vous desirez comme le nom du module ou sa vitesse. Assurez vous que vous avez choisi une vitesse qui fonctionnera avec votre carte et cliquez [Sauvez cfg]. Cela sauvegardera la configuration. Si il y a une erreur pendant la sauvegarde celle ci est interrompue

Notez qu'il faudra refaire les memes etapes pour chaque module bluetooth que vous devez configurer
