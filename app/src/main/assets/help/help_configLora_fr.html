Configuration d'un module eByte Lora E220

De manière à pouvoir effectuer de la télémétrie longue distance vous devez remplacer votre module bluetooth par un module télémetrie ngue portée comme le Lora E22O.

Les modules ci-dessous ont été configurés avec succé

E220-400T30D

E220-400T30S

E220-900T30D

E220-900T30S

E220-400T22D

etc ...

Pour programmer votre module il est necessaire d'avoir un cable TTL

un adaptateur USB (USBC ou micro USB en fonction de votre apareil Android )

un adaptateur jst et des cables jst

A noter que la batterie de votre appareil Android doit être suffisamment puissante pour alimenter le module Lora. Envisager l'utilisation d'une batterie externe si vous avez un module puissant.

De manière à mettre le module en mode configuration il faut mettre les 2 cavaliers

Vous devez configurer votre module de manière à ce qu'il puisse fonctionner avec votre altimètre.

A noter que ces modules sont en général préconfigurés à 57600 bauds. Cependant votre altimètre par défaut sera configuré à 38400 bauds vous devez donc vous assurer de vous connecter à la bonne vitesse.

A noter que les modules Lora sont vendu par paire donc il sera nécessaire de configurer à la fois l'émetteur et le récepteur

ATTENTION: assurez vous de ne pas débrancher le module lorsque vous faites les opérations qui suivent

Charger la configuration du module

Branchez le module dans votre appareil Android en utilisant un câble OTG ou un adaptateur USB qui peut fonctionner avec micro USB ou USB type C.

Ensuite cliquez sur [charger cfg] , cela effectuera des tentatives de connexion au module à 9600 baud. Puis la configuration de module sera chargée de manière à ce que vous puissiez la changer.

Sauvegarder la configuration du module

Change the settings that you want to change. Changez les paramètres que vous voulez changer. Assurez vous que vous avec choisi la vitesse qui fonctionnera avec votre carte (par défaut 38400). Assurez vous que les channels sont les meme pour l'emetteur et le recepteur. Si vous utilisez une clé d'encription assurez vous qu'elle soit la meme des 2 cotés. Lorsque terminé cliquez [Sauver cfg]. Cela sauvegardera la configuration. En cas d'erreur lors de la sauvegarde celle-ci sera annulée

Note that you will need to repeat the same step for each module

Avant de l'utilisez assurez vous d'enlever les 2 cavaliers pour sortir du mode configuration

Lora telemetry modules currently do not work with ESP32 board.
