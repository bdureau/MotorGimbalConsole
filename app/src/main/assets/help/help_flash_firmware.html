Flash firmware help file

This is a quick help file help you to flash the latest Motor Gimbal Firmware. In order to work correctly, the Gimbal firmware need to be compatible with the Gimbal console application


In order to flash the firmware you will need to connect the Gimbal to your android device using a USB cable and an adapter

First you need to select the firmware. The firmware depend on the gimbal that you want to flash

Then decide if you want to directly flash your firmware or use a recovery firmware first

Use a recovery firmware if the config is wrong and the gimbal can no longer connect.

After using a recover firmware you will need to flash the firmware.

Note that flashing the wrong firmware will result in a gimbal not working at all or not working as expected

In order to be able to flash your gimbal you will need a USB cable and an adapter to plug it to your phone.

a USB adapter (USBC or micro USB depending on your Android device)

some jst cables are required

The following gimbal can be flashed, please make sure that your carefully look at the pictures to identify your gimbal

Gimbal (with an MPU6050) (Large version)

Gimbal-bno055

Flashing STM32 based altimeters (ie: AltiMultiSTM32 and AltGPS)

- Plug the cable

- Select the correct firmware or detect it

- Move the boot0 jumper to the right

- Press the reset button

- Press the [flash firmware] button

- Move the boot0 jumper back to the left

Firmware should be loading, when complete press the reset button

 

IMPORTANT NOTE: when flashing an altimeter do not disconnect the cable while flashing, this will corrupt the firmware and you will have to flash it again
