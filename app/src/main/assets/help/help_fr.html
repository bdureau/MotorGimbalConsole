Fichier Aide de la Console Gimbal Bear Altimeter

Ceci est une aide rapide pour utilser votre Gimbal Bear altimeter sur Android. La console Android est très pratique à utiliser pour reconfigurer votre Gimbal lorsque vous etes sur le terrain.

Tout d'abord vous devez vous assurer que votre Gimbal est compatible (c'est à dire que le bon firmware a été chargé).

Ensuite assurez vous que le module bluetooth est connecté au Gimbal. 

Puis vous devez aller dans la configuration de votre appareil Android et dans la configuration bluetooth chercher les module bluetooth HC-05, HC-06 ou HC-07.

Lorsque vous l'avez trouvé entrez le code pour associer le module. Le code par dèfaut est 1234.

!!! ATTENTION !!!!

Il est souhaitable de nommer clairement chaque adaptateur bluetooth sur votre téléphone de manière à vous assurer que vous configurez le bon Gimbal. 

Ensuite allumer le Gimbal et attendre jusqu'à ce la fin de l'initialisation de celui-ci.

Puis démarrer l'application sur votre appareil Android, cliquer sur montrer les périphériques bluetooth et choisissez le module HC-05 ou HC-06. L'application tente ensuite de se connecter au Gimbal.

Si le message connecté apparait, cela veut dire que vous etes connecté et que vous pouvez utiliser la Console pour configurer votre  Gimbal ou lire vos vols. Si cela n'arrive pas à se connecter assurez vous que vous avez selectionné le bon module bluetooth ou que le Gimbal est sous tension.

Utilisation de la console du Gimbal

Ecran principal du Gimbal

Sur l'écran principal vous avez un certain nombre de boutons, décrivons les. 

[Configuration Gimbal]

Charge la configuration de l'altimètre et vous permet de la modifier.

[Vols]

Charge la totalité des vols que vous avez enregistré et les affiche.

[Reinitialisation]

Donne la possibilité de restaurer les paramètres d'usine de la configuration du Gimbal. Cela permet également d'effacer tous les vols. 

[Flash Firmwares]

Donne la possibilité de flasher le Gimbal avec le dernier firmware pour etre certain que celui ci soit compatible avec la console Gimbal

 

[Connexion/Déconnexion]

Se connecte ou déconnecte du Gimbal

Menu principal du Gimbal

Depuis le menu vous pouvez accèder à ce fichier d'aide, un écran à props qui vous donnera des informations concernant la console Gimbal et sa version . Vous avez égalment la possibilité de changer l'apparance de l'application.

 

 

La console est compatible avec les Gimbals suivant:

- Gimbal (MPU6050)

- Gimbal (BNO055)
